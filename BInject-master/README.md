# bInject

一个带界面的，支持向进程注入DLL或注入汇编代码的工具

## 目前支持功能
- 进程信息查看
- 进程挂起
- 进程恢复
- 进程模块查看
- 进程模块卸载
- DLL文件拖拉
- DLL注入
- 根据汇编代码生成OPCode(目前只支持x86)
- 直接注入汇编代码


## 编译环境
- VS215
